# Linear Equation Solver (C++)

## Description
This project is a C++ program that solves systems of linear equations with multiple variables.

## Features
- Supports up to 100 equations
- Handles floating point values
- Accepts equations with spaces
- Simplifies equations
- Solves using Gaussian Elimination

## How it Works
1. Reads equations
2. Parses variables and constants
3. Builds matrix
4. Applies Gauss-Jordan elimination
5. Outputs solutions

## Example Input
3
2x1 + 3x2 - x3 = 10
x1 - x2 + 4x3 = 5
-3x1 + 2x2 + x3 = 7

## Output
x1 = 0.678
x2 = 3.535
x3 = 1.964

## Technologies
- C++
- Vectors
- Parsing
- Linear Algebra
